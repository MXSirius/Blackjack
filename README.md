# Blackjack Python Game

A simple implementation of the classic Blackjack card game using Python.

## Features
- Classic Blackjack gameplay mechanics
- Game state saving and loading
- Simple text-based user interface

## Installation
1. Ensure Python 3.x is installed on your system
2. Clone this repository
3. Navigate to the Blackjack directory
4. Run the game with: `python blackjack.py`
